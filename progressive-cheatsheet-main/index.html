html Elements

Element Description Example
<head> Contains metadata/information for the document. Here you need to connect stylesheet (see <link> below) and <title>for the browser or tab.
<link> Defines the relationship between a document and an external resource (most used to link to style sheets) <link href="../project-name/styles.css" rel="stylesheet">
<header> Defines content meant to be at the head of the site (nav/headings/opening paragraphs etc) <header>
<nav>
<p>
<nav> Defines navigation links <nav>
<a href="/html/">HTML</a> |
<a href="/css/">CSS</a> |
<a href="/js/">JavaScript</a> |
<a href="/python/">Python</a> </nav>
<h1>-<h6> Headings Fairly self explanatory- Most important heading is #1
<a> Anchor elements are used for Hyperlinks. <a href="https://www.w3schools.com">This is a link</a>
<img> HTML images are defined with the <img> tag. The source file (src), alternative text (alt), width, and height are provided as attributes <img src="w3schools.jpg" alt="W3Schools.com" width="104" height="142">
<span> Defines a section in a document (in content style change) <p>My mother has <span style="color:blue">blue</span> eyes.</p>
<!--> comment that only shows up in the html file that wont show up in the browser. <!--This is a comment. Comments are not displayed in the browser--> <p>This is a paragraph.</p>
<footer> defines the content meant to be at the bottom, can be more nav, social links, or just a closing paragraph
